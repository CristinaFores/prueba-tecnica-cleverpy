# 📑 Cleverpy App

> Web application represent the returned posts

 </br>

# 💫Getting Started

**Install NPM packages**

```sh
npm install
```

**Run the project**

```sh
npm start
```

- Then will open [http://localhost:3000](http://localhost:3000)

<br/>

**Login Credentials **

```sh
- Usuario: admin
- Contraseña: admin
```

<br/>

**Run the test**

```sh
npm test

npm run cypress:open
```
